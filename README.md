# homebrew-tap

Homebrew formulae for tools by [Abdelrahman Awad](https://github.com/logaretm).

## Installation

```sh
brew tap logaretm/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [depnuke](https://github.com/logaretm/depnuke) | Nuke node_modules, lock files, and package manager caches |

### depnuke

```sh
brew install logaretm/tap/depnuke
```

Or after tapping:

```sh
brew tap logaretm/tap
brew install depnuke
```

## Updating

Formulae are updated automatically on each release. To upgrade:

```sh
brew update
brew upgrade depnuke
```
